# streamlit-dnd-sortable

Streamlit component: vertical drag-and-drop reorderable list (**@dnd-kit**).

## Dev workflow

From `streamlit_dnd_sortable/frontend`:

```bash
npm install
npm run build
```

Set `_RELEASE = False` and `npm start` (port **3010**) in `streamlit_dnd_sortable/__init__.py` for React hot reload against Streamlit dev.
